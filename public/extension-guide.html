Installer la synchronisation des cases

  1. Dans Chrome, ouvrez chrome://extensions
  2. Activez Mode développeur en haut à droite.
  3. Cliquez sur Charger l’extension non empaquetée.
  4. Sélectionnez ce dossier :C:\Users\proda\OneDrive\Documents\LEGO\extension
  5. Si le module était déjà installé, cliquez d’abord sur son bouton Actualiser dans chrome://extensions.
  6. Ouvrez ou rechargez n’importe quelle page Rebrickable. Le module ouvrira la part list 108467 dans un onglet inactif, réalisera l’export puis fermera cet onglet automatiquement.
  7. Pour un set comportant inventory=1 ou une autre révision, le même mécanisme exporte automatiquement la version exacte avant la recherche.
  8. Pour un lien /mocs/MOC-…, le module exporte également l’inventaire du MOC avant de calculer le plan.

Le module utilise uniquement Export Parts → Rebrickable CSV dans des onglets inactifs. Il reproduit le formulaire d’export, sans parcourir les pages de pièces et sans accéder à votre mot de passe.
